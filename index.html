Portfolio con diferentes proyectos e ideas

Aplicación para la gestión de citas de una peluqueria.

Frontend: Registro de usuarios, login de acceso al front para poder elegir un día y hora del calendario. Uso de la libreria de Full Calendar para la visualización del calendario.

Backend: Control de usuarios registrados, citas registradas y control o modificación de citas. Desde el backend el administrador tiene total control del registro y datos de los clientes, así como la opción de aceptar o cancelar citas elegidas por el cliente.

Proyecto realizado con React + Supabase

Despliegue con Vercel
