Portfolio con diferentes proyectos e ideas

Aplicación para la gestión de citas de una peluqueria.

Frontend: Registro de usuarios, login de acceso al front para poder elegir un día y hora del calendario. Uso de la libreria de Full Calendar para la visualización del calendario.

Backend: Control de usuarios registrados, citas registradas y control o modificación de citas. Desde el backend el administrador tiene total control del registro y datos de los clientes, así como la opción de aceptar o cancelar citas elegidas por el cliente.

Proyecto realizado con React + Supabase

Tienda online Venta camisetas de fútbol Fancrew.

Frontend: Registro de usuarios, login de acceso al frontend por parte del cliente. Visualización de productos, posibilidad de añadir producto al carrito y realizar el proceso de compra, incluido el pago desde la web (aunque no es la mejor práctica en un entorno real y si reedirigir a una plataforma como pudiera ser Redsys).

Backend: Acceso por parte del administrador. Control de usuarios registrados, compras registradas y control o modificación de categorias, tags, productos y stock. Desde el backend el administrador tiene total control sobre la tienda, donde podrá crear, actualizar, modificar o eliminar cualquier producto, imagen,etc. El administrador tendrá una visual general del stock disponible en la tienda, con el buscador realizado para esto.

Proyecto realizado con JS, HTML, CSS, PHP + MySQL + Docker
